# Application-management-system
A web built with Node.js, Express, MongoDB, Handlebars. the system manages job applications.
It supports full CRUD functionalities for candidates, application,and interview notes, along with a simple UI.

## Features
- full CRUD functionalities(Create, Read, Update, Delete)
- Manage candidates with unique email validation
- Track application with status filtering
- Add and manage interview notes with ratings
- Sever rendered UI using Handlebars
- summary dashboard with aggregated insights 
- Input validation and error handling

## Tech Stack
- Node.js
- Express 
- MongoDB
- Handlebars
- CSS
- Jwt
- Nodemailer
